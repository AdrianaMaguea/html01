NOTAS 
FORMULARIOS 

Utilizar etiqueta <form> para empezar un formulario 
 Etiqueta <input> para abrir un caja de texo
 Atributo TYPE para decir que tipo de entrada queremos tener (ejemplo: TEXT) 
 Atributo PLACEHOLDER para mostrar un texto predeterminado dentro de la caja pero que se borre automaticamente al empezar a escribir 
 Atributo MAXLENGTH numero de caracter maximo que queremos en nuestra caja 
 Atributo NAME para decir al navegador que estamos pidiendo el nombre 
 Atributo ID para al momento de ponerle nombre a la caja con la etiqueta <label for> decirle que ese texo es correpondiente a ese input llamado "nombre" 
 (el puntero al momento de clicar el nombre de la caja nos lleva a la caja de texo)
